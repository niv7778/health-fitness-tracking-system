# health-fitness-tracking-system
## Overview
This project is a no-code / low-code Health and Fitness Tracking System designed to log workouts, daily activities, and personal fitness goals. The system automates data organization and user association without traditional programming.

## Tools Used
- Airtable (Database & Automations)
- JotForm (Data collection forms)
- Notion AI (Documentation)
- ChatGPT (Workout categories & suggestions)
- Bubble (UI design - conceptual)
- Make & n8n (Workflow automation - conceptual)
- Nanobanana (Gemini) (Visual analytics)

## System Architecture
User → JotForm / Bubble → Airtable → Automations → Reports & Insights

## Features
- Workout logging
- Daily activity tracking
- Goal management
- Automated user-record linking
- Structured no-code database design

## Implementation Notes
This project focuses on no-code implementation using Airtable automations and form-based data input. External workflow tools are included as part of the system architecture and documentation.
